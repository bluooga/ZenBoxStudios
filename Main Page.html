Info

Zen Box Studios is a indie game dev group created from the remains of Zen Arc Studios. in this group we create Games with our upcoming game Moonlight set to release in march 2020

Zen Arc Studios Shut Down

Up until now the only reason for Zen Arc's sudden shut down has been that it was not sustainable, while that remains true the underlying reason is actually due to a lawsuit by Studio Lico, since Zen Arc was a non profit group it was not possible to pay the fine. again we do not wish for any hate towards studio lico as their work is amazing and we did infringe on their copyright without an official license.
